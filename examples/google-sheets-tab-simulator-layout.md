# Google Sheets Tab Layout For Simulator Imports

## Suggested Tabs

1. `SIM Import - Proxy 10M`
   Use this as the main working tab.

2. `SIM Import - Aggregate`
   Use this only for high-level sanity checks.

3. `SIM Import - Notes`
   Optional notes tab for assumptions and mapping rules.

## Files

- `google-sheets-tab-simulator-proxy-10-member.csv`
- `google-sheets-tab-simulator-aggregate.csv`
- `params-tab-simulator-proxy-10-member.csv`
- `params-tab-simulator-aggregate.csv`
- `params-tab-simulator-notes.md`

## Recommended Workflow

- Open the `google-sheets-tab-simulator-proxy-10-member.csv` file in Google Sheets as a new tab.
- Keep rows 1-4 as notes/header helpers.
- The exact simulator import header starts on row 5.
- For direct simulator import, use `params-tab-simulator-proxy-10-member.csv`.
